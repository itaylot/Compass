# Design brief: נורווגיה 2026 (family trip companion)

- **Design read:** a Hebrew-speaking family (parents included) planning and living a 13-day Norway road trip; the register is warm, calm, trustworthy, "quality travel app", never spreadsheet.
- **Concept spine:** "המפה היא התוכנית" (the map IS the plan). The itinerary list and the map are two views of one object; selecting anywhere selects everywhere.
- **Delivery tier:** editorial (daily-use tool; must feel instant; micro-motion only). This is site class (c), a product tool, `type: "website"`.
- **Locked palette:** warm paper `#F6F3EC` background, ink `#24313A`, fjord teal `#17606A` primary, glacier blue `#2E7DA6` secondary, berry `#C25E8A` highlight, per-category functional colors for pins/chips. Defense: cool fjord tones on warm paper reads Nordic-outdoors without the banned dark+neon or cream+brass families.
- **Locked type:** Heebo (400/500/600/700/800) for all Hebrew UI; a geometric Hebrew-native sans keeps large day numerals and dense table text equally readable. No serif (no brand justification for one).
- **Tier-1 signature interaction:** linked map↔timeline selection (the app's core mechanic per the user's explicit spec): tapping a pin highlights and scrolls the matching timeline card; tapping a card flies the map to its pin. Micro-motion: pin scale ring, card highlight pulse, toast on status change.
- **Screen plan (app translation of sections):** (1) היום: day summary card + day picker + mini map + chronological timeline; (2) מפה: full map, category filters, bottom info card; (3) תכנון: mobile Sheets-like table with sort/filter; (4) מידע: bookings, EV, emergency, packing, quick links, "מה בא לנו עכשיו". Each screen has one layout family; no fake data surfaces, all state is real.
- **Asset plan (explicit user constraint):** the user's spec mandates emoji category glyphs (🏨🍽️🥾📍⚡🚗🎟️) as the icon system; hand-authored SVG app icon/favicon (fjord mark) + OG text values. No generated photography: a POC data tool, zero-credit workspace, and the spec asks not to overload.
- **CTA inventory:** "נווט" (outline pill, opens Google Maps); "סמן שבוצע" (soft-fill pill, mutates status + toast); tab bar items (icon+label, active tint); day chips (pill, active fill); filter chips (tinted per category); "פתח ב־Google Sheets" (ghost button, demo toast); "מה בא לנו עכשיו" mood buttons (large tiles). Each is its own component styling, no shared button utility.
